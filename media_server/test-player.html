📝 Quick Guide

  1. Start streaming from Larix Broadcaster or OBS to: rtmp://your-server:1935/live/test
  2. Enter your server URL above (e.g., http://localhost:8002)
  3. Enter your stream key (e.g., "test")
  4. Click Play Stream to watch

🎬 Example Stream URLs:

  • RTMP Publish: rtmp://localhost:1935/live/test
  • HTTP-FLV Play: http://localhost:8002/live/test.flv
  • HLS Play: http://localhost:8002/live/test/index.m3u8
